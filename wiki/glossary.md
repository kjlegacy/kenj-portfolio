# Glossary

Definitions of key terms used in this project.

### Bento Grid
A design layout style characterized by rectangular tiles of varying sizes, popularized by Apple and modern dashboard designs.

### LLM Wiki
A documentation system maintained by Large Language Models, designed to stay in sync with a changing codebase via automated agents.

### Glassmorphism
A design style using background blur and transparency to create a "frosted glass" effect.

### Confidence Score
A metric used in the wiki to indicate how likely the documentation matches the current state of the source code.
- **High**: Recently verified against the exact source commit.
- **Medium**: Source files have changed slightly since last verification.
- **Low**: Source files have changed significantly; re-verification needed.
